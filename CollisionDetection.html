3D Collision Detection Optimization and Implementation

Github

In order to increase my familiarity with game physics, I decided to implement and optimize a collision detection system. This project was implemented under PXG3D, a custom game engine I made. The project took 3 weeks to develop and has the following features:

1. Collision Detection Implementation

Any GameObject can have a ‘PhysicsComponent’ where a Collider can be assigned to. Given 2 convex colliders, PXG3D can detect if they are overlapping using the seperating axis test.

2. Collision Detection Optimizations

In order to optimize this process, I decided to implement an optimization that were mention in this GDC talk.


The first optimization relates to optimizing the checking of face normals for a separating axis, which allows you to only need to go through the faces of one polyhedron instead of both of them. Because of this, in the case where both polyhedrons have the same amount of faces, the optimization would make the process twice as fast.


The second optimization relates to optimizing the process checking the edges on one polyhedron and the other polyhedron to find a separating axis. The brute force method of doing this is by checking each edge of one polyhedron with every other edge in the other polyhedron. Instead, this optimization focuses on only checking the edges that can actually create a valid separating axis.
